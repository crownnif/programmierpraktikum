Projekt: Bildbearbeitung

In diesem Projekt können Algorithmen zur Bildbearbeitung implementiert
werden. Einige sind vorgegeben, andere sollen hinzugefügt werden.
Es können Bilder geladen, dargestellt und manipuliert werden.

Die Klasse SWBild definiert ein einfaches Bildformat fuer Schwarz-Weiß-Bilder
sowie die Operationen auf diesem Bildformat.
    
Startpunkt: Erzeuge ein Exemplar der Klasse SWBild über den Konstruktor,
der einen Parameter bekommt. Dieser Parameter ist ein Dateiname für
eine Bilddatei. Im Projektverzeichnis liegt ein Unterverzeichnis
'bilder' mit einigen Bildern, die bearbeitet werden können. Als Parameter
kann also beispielsweise "bilder/kiss.gif" angegeben werden und
das entsprechende Bild wird geladen.

Alternativ könnt ihr auch den parameterlosen Konstruktor benutzen, um mit 
einem interaktiven Dateidialog eine Bilddatei auszuwählen.
    
In diesem Projekt soll der Umgang mit Schleifen und Arrays geübt werden.
